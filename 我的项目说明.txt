1.配置
git config --global user.name "       "
git config --global user.email 
2.本地库初始化 git init
3.查看状态： git status(查看工作区、暂存区的状态)
4.添加操作: git add 文件名(将工作区新建/修改的内容添加到暂存区)
5.提交操作： git commit -m “commit message” 文件名(将暂存区的内容提交到本地库)
6.查看历史记录
(1)git log
(2)git log --pretty=oneline
(3)git log --oneline
(4)git reflog 
7.查看本地库中的所有分支
git branch -v 
8.创建一个新的分支
git branch dev 
9.切换分支
git checkout dev 
10.切换到接收修改的分支
git checkout master
11.执行merge命令
git merge dev
